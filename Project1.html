Lesson 1: Basics of the Web and HTML

Basics of the Web

The world wide web is made up of network of communicating computers and servers which share data through the use of HTTP (Hyper text Transfer Protocol), a protocol used by the web to request information that is stored on a server through an HTTP Request from a browser on a personal computer. The browser then recieves information from the server, and interprets the information to display to the user.

HTML

HTML, also known as Hyper Text Markup Language, is the language used by the web to describe to browsers what users should see, in the form of text content, and what it should look like, which is markup. HTML can also reference other documents, such as videos and music, and it can also provide links to other pages, in the form of hyperlinks.

Tags and Elements

An HTML document is made up of elements, which modify text content, provide links to other HTML documents, or reference other documents. These elements are comprised of tags, which tell the browser what type of element is being used. Some examples of tags include emphasis (em), bold (b), anchor (a), image (img), break (br), and paragraph (p).

These tags may also contain attributes, which adds a value to the tag. For example, the anchor tag uses the href attribute, where the value references a url which links the content of the element to another page.

As the computer interprets the instructions from the HTML document literally, it can be thought of as stupid. This is because a small mistake in the language created by the programmer could cause huge problems, as the computer might not be able to interpret it as it can only follow strict instructions.

Inline vs. Block Elements

HTML elements are either block or inline. Block elements form a box around the content, which separates it from the rest of the text, while inline elements are in the general body of text, and do not have the box surrounding them.

Lesson 2: Creating a Structured Document with HTML

Developer Tools (in the browser)

Some browsers contain developer tools like DOM, "Developer Object Mode", which allows developers to visually view the "tree-like structure" of the HTML and the effect of the CSS on style for the blocks, or boxes.

The "Tree-Like Structures" of HTML

The "tree-like structure" of HTML is that HTML elements often have other HTML elements of them with more HTML elements inside of them and so on. This creates relationships between and within elements with can look like a family tree, hence the "tree-like structure".

The Relationship between HTML and Boxes

When viewing an HTML document, you can often see a "wave" occuring due to the tags. This can be used to see the HTML elements, or boxes, and how they are nested inside each other and how they relate to each other. It can also show how deeply nested a box is, based on how far the indentation is for its corresponding code.

Text Editors (for programming)

For viewing the "wave" mentioned in the previous section, it is made a lot easier by special text editors (like Sublime Text) which are specially made for programming. These editors automatically indent based on how much a box is nested, and can also generate a closing tag when an opening tag is written into the code.

Lesson 3: Adding CSS Style and HTML Structure

Avoiding Repition

Avoiding repition in coding is important for two major reasons, avoiding making errors and ability to easily change style. By avoiding repeating code, programmers can reduce the chance that will be an error in one of the repeated elements. This also helps as it decreases the amount of time a programmer is looking for the element or elements causing problems. By having the style choices in a single place instead of repeated, it becomes a lot easier to change the style choices.

CSS

CSS, or Cascading Style Sheets, is coding that controls the style of the HTML elements. This is done by coding styles in the CSS document to different classes that are used by elements the HTML document. All of the elements with the class will then have the same style applied to them.
